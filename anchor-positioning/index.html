CSS Anchor Positioning

Three patterns — no JavaScript position sync, no position: relative wrappers.

Pattern 1

Tooltips

Each tooltip is a flat sibling of its button — no wrapper with position: relative needed. position-try-fallbacks: flip-block auto-flips tooltips that would overflow the viewport edge.

Pattern 2

Form Validation

The error messages live in a separate <div> outside the <form> entirely — scroll down to see it in the source. Anchor positioning places them below their inputs regardless.

Pattern 3

Dropdown

The Popover API wires open/close with zero JavaScript. Anchor positioning places the menu below the button regardless of scroll position or DOM location. Press Esc to dismiss.
