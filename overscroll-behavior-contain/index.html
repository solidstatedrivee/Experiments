overscroll-behavior: contain prevents scroll chaining — when a nested scroll container reaches its boundary, the parent does not scroll.

Example 01

Toggle Comparison

Flip the switch to remove overscroll-behavior: contain at runtime via a class swap. In the unchained state, scrolling past the top or bottom of the message list chains to the page scroll.

Contained — overscroll-behavior: contain contain 
/* Default: containment on */
.chat-panel { 
    overscroll-behavior: contain; 
}

/* After toggle: class "unchained" added */
.chat-panel.unchained { 
    overscroll-behavior: auto; 
}

Example 02

Chat Panel

The message list is fixed-height and scrollable inside a page that also scrolls. Without overscroll-behavior: contain, reaching the top or bottom of the conversation would chain scroll to the page. With it applied, scrolling stops at the boundary.

.chat-panel {
    height: 300px;
    overflow-y: auto;
    overscroll-behavior: contain;
}

Example 03

Modal with Scrollable Body

A long-form modal with a scrollable content area. With overscroll-behavior: contain on the modal body, scrolling to the end of the text and continuing does not scroll or rubber-band the background page. 

.modal-body {
    max-height: 60vh;
    overflow-y: auto;
    overscroll-behavior: contain;
}

Example 04

Horizontal Card Carousel

The card row scrolls horizontally inside a vertically-scrolling page. overscroll-behavior-x: contain prevents swiping past the last card from triggering the browser's back/forward navigation gesture on macOS and iOS. 

.carousel {
    display: flex;
    overflow-x: auto;
    overscroll-behavior-x: contain;
}
